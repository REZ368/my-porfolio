FSIS (Fruit Stock Inventory System)

Is a personal JAVA project for TESDA developed to manage and monitor fruit inventory efficiently. The system is designed to handle the basic inventory tasks such as recording terms, tracking incoming fruit supplies, managing pricing, and monitoring quantities.

Technologies used: Java used for building the front-end interface and functionality. MySQL serves as the database for storing all inventory-related data securely and efficiently.

Users:
Add New Fruit :Users can register new types of fruits into the system with relevant details like name, price, and quantity.
Track Incoming Supply: The system allows the entry of incoming fruit batches, updating the total quantity and maintaining accurate stock levels.
Price and Quantity Monitoring: FSIS keeps track of the price per fruit and its available quantity for inventory and sales reference.

FSIS aims to be a lightweight and straightforward inventory tool, ideal for small-scale fruit vendors, market stalls, or personal use.
